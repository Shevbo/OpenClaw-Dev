Голос вместо чата (веб)

Удерживайте кнопку, говорите, отпустите — фраза уходит на сервер: Google Speech-to-Text → текст ответа (заглушка или ваш AGENT_URL) → Google Text-to-Speech (MP3) → воспроизведение в динамиках. Телеграм и телефония не используются.

Локальная отладка: статика на порту 8765, API на 8091 — откройте с параметром ?api=http://127.0.0.1:8091. За Caddy — один хост: см. scripts/Caddyfile.voice-mvp.snippet. Старый тест WebSocket: legacy-ws-mic.html.

Голосовой диалог (не Telegram voice)

Эта страница открывается из Telegram по кнопке и даёт браузерный доступ к микрофону. Полный дуплекс с агентом подключается к медиа-каналу шлюза OpenClaw (Voice plugin / WebRTC), когда в конфиге задан URL WebSocket. Пока URL не задан, идёт только проверка микрофона и статус соединения.
